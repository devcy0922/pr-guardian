# 보안 체크 기준

## 절대 금지 (즉시 차단)
- API 키, 토큰, 비밀번호 하드코딩
- `.env` 파일 커밋
- 개인키(`.pem`, `.key`) 파일 커밋

## 검사 대상 패턴
- AWS 키: `AKIA`로 시작하는 20자 문자열
- OpenAI 키: `sk-` 접두사
- 데이터베이스 URI에 비밀번호 포함
- Basic Auth 헤더 하드코딩

## 보안 코딩 원칙
- 입력값 검증 (SQL injection, XSS 방지)
- 환경 변수를 통한 설정 주입
- 최소 권한 원칙
- 에러 메시지에 내부 정보 노출 금지
