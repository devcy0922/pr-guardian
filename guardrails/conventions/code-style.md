# 코드 스타일 가이드

## 공통
- 들여쓰기: 2 spaces (JS/TS), 4 spaces (Python)
- 파일 끝 빈 줄 1개
- console.log 디버깅 코드 금지 (프로덕션)
- TODO 주석은 이슈 번호 포함 필수

## TypeScript
- strict 모드 필수
- `any` 타입 금지
- 함수 반환 타입 명시
- 인터페이스 이름은 PascalCase

## Python
- type hint 필수
- docstring (Google style)
- f-string 사용 권장
- 함수명은 snake_case
